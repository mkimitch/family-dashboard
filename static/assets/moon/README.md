# Moon Phase Icons

These icons are derived from NASA Scientific Visualization Studio's "Moon Phases Loop" frame sequence, SVS ID 4310.

Credit: NASA's Scientific Visualization Studio. Visualizer: Ernie Wright (USRA).

The source TIFF frames were processed for this project by cropping to a square, removing the black background, resizing, and exporting as transparent PNG files.

# Third-Party Notices

## Moon phase icons

This project includes processed moon phase icon assets derived from NASA Scientific Visualization Studio imagery.

Source:

- Title: "Moon Phases Loop"
- Source: NASA Scientific Visualization Studio
- SVS ID: 4310
- Visualizer: Ernie Wright (USRA)
- Mission/data context: Lunar Reconnaissance Orbiter (LRO); LOLA and LROC datasets
- Original frame sequence: `frames/5760x3240_16x9_30p/moon.0001.tif` through `moon.0234.tif`

Modifications made for this project:

- Cropped to a 1:1 square composition
- Removed the black background
- Resized to transparent PNG icons
- Mapped to OpenWeather `daily.moon_phase` values
